# InAmigos Foundation Website

This is a simple NGO awareness website created using HTML and CSS. The website shares information about InAmigos Foundation, its projects, and the positive impact it is making in society.

## Features

* Home page with hero section
* About the NGO
* Information about different projects
* Image gallery
* Call-to-action buttons
* Responsive design for different screen sizes

## Technologies Used

* HTML5
* CSS3

## Purpose

The purpose of this project is to spread awareness about the work of InAmigos Foundation and encourage people to support or volunteer for the NGO.
